## LegiStar Quick Template

Since 20+ cities use LegiStar, here's the pattern:

```javascript
// Base URL: https://[city].legistar.com/Calendar.aspx
// Event list: #ctl00_ContentPlaceHolder1_gridCalendar_ctl00
// Event item: tr.rgRow or tr.rgAltRow
// Name: td:nth-child(2) a
// Date: td:nth-child(1)
// Time: td:nth-child(3)
// Location: td:nth-child(4)
// Details: td:nth-child(2) a[href]
```

Just change the city name in the URL!
